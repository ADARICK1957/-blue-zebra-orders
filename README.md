# Blue Zebra Officials Prep Card Order Website

A simple static website for taking Blue Zebra Officials Prep Card order requests.

## What it does

- Shows the product and pricing
- Calculates order total by quantity:
  - 1–49 cards: $9.95 each
  - 50–99 cards: $6.95 each
  - 100+ cards: $5.95 each
- Uses a simple email-based order form
- Lists Venmo/check as payment options
- Mentions that card payment can be added later with Square, Stripe, or PayPal

## How to use

1. Upload these files to a GitHub repository.
2. Turn on GitHub Pages for the repository.
3. Replace the email address in `index.html` and `script.js` if needed.
4. Replace the card preview with a real card image when ready.

## Files

- `index.html`
- `styles.css`
- `script.js`

## Important note

This first version does not process credit cards. That is intentional. It avoids monthly software costs and lets Rick confirm orders before payment.
